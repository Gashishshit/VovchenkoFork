# DETR-based Ore Contamination Detection System

Система детекции рудозасорения на конвейерных лентах на основе трансформерной архитектуры DETR (Detection Transformer) с графическим интерфейсом для инференса.

##  Возможности

- **Обучение**: Полный пайплайн обучения DETR модели на кастомном датасете
- **Инференс**: CLI и GUI интерфейсы для детекции на изображениях
- **Batch обработка**: Пакетная обработка множества изображений через GUI
- **Визуализация**: Отрисовка bounding boxes с указанием классов и confidence score
- **Метрики**: Подсчет FPR, Recall, Precision на валидационном наборе
- **Аугментации**: Специализированные аугментации для промышленных условий

Установка зависимостей
pip install -r requirements.txt

Скачайте веса обученной модели и поместите их в checkpoints/best_model.pth

GUI Приложение
python main.py
